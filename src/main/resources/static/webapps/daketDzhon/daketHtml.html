Для создания таблиц используют тэг TABLE

Формы в HTML

Для создания формы необходимо ввсести блочный тэг form -в формы вводят данные и можно отправлять на сервер и ....
Атребут action="" указывается адрес документа который будет обрабатывать запрос в JAVA SERVLET
Атрибут method="" принемает два значения POST and GET
Атребут name - задает имя формы
Атребут novalidate или formnovalidate - позволяет не проверять на правельность заполненности
Атребут autocomplete="" принемает два значения ON или OFF- в случае обновления страниции вкл. атребут позволяет не вводить повторно введенный ранее текст, текст будет сохраняться в форме
Атребут enctype="" или formenctype=""- указывает тип кодирования работает с методом POST принемает 3 значения "application/x-www-form-urlencoded"-кодирует текстовые поля (по умолчанию) "multipart/form-data"-нужно для отправке изображения или файлов данные не кодируются "text/plain"-данные не кодируются пробелы заменяются знаком +
TEG input - создает форму для ввода текста, type- text,number,date ...., name- название формы ввода
Атребут Target -указываем куда будет загружаться возвращенный с сервера запрос

Семантика

Тэг div блочный тэг
Тэг Header - тэг для создания шапки
Тэг Aside - тэг для создания сайд бара
Тэг Section- тэг для создания повторяющегося контента на странице(может содержать несколько article)(для контейнера всей странице лучше использовать div)
Тэг Footer- тэг для создания подвала сайта
Тэг Nav- тэг для создания навигации по сайту
Тег figure используется для группирования любых элементов, например, изображений и подписей к ним.
Тег figcaption содержит описание для элемента figure. figcaption должен быть первым или последним элементом в группе.
Тег article используется в качестве контейнера для любого независимого раздела страницы.
Тег main предназначен для основного содержимого документа. На странице может быть только один main и он не должен располагаться внутри элементов article, aside, footer, header или nav. Элемент включает в себя содержимое, которое является уникальным для данного документа, и не должен включать повторяющиеся разделы сайта, такие как навигация, название сайта, логотип, поисковая форма.
Тег menu предназначен для отображения списка пунктов меню (label — Устанавливает видимую метку для меню.)(type context — Создаёт контекстное меню.toolbar — Меню в виде панели инструментов. list — Меню в виде списка.
Тег label устанавливает связь между определённой меткой, в качестве которой обычно выступает текст, и элементом формы (input,select,textarea). Такая связь необходима, чтобы изменять значения элементов формы при щелчке курсором мыши на текст. Кроме того, с помощью label можно устанавливать горячие клавиши на клавиатуре и переходить на активный элемент подобно ссылкам.Существует два способа связывания объекта и метки. Первый заключается в использовании идентификатора id внутри элемента формы и указании его имени в качестве атрибута for элемента label.

Тег audio добавляет, воспроизводит и управляет настройками аудиозаписи на веб-странице. Путь к файлу задаётся через атрибут src или вложенный элемент source. Для универсального воспроизведения в браузерах аудио кодируют с помощью разных кодеков и добавляют файлы одновременно через элемент source.
