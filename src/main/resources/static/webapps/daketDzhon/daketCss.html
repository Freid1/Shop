Макет

position: static- каждый последующий блочный элемент на странице располагается ниже, чем предыдущий.
position: relative.(top, bottom, left или right)При использовании данной схемы элемент перемещается относительно места, в котором он бы находился при использовании схемы static (position : relative; top : lOpx ;)
position: absolute- блок исключается из нормального потока и не влияет на положение других элементов страницы. (top, bottom, left или right)
position:fixed- При использовании данной схемы элемент позиционируется относительно окна браузера. Таким образом, при прокрутке страницы элемент остается на том же месте
z-index Если вы хотите задать, какой элемент будет находиться на переднем плане. Его значением является число, и чем выше это число, тем ближе данный элемент к переднему плану.(использовать с POSITION)
Свойство float позволяет поместить элемент, позиционированный с помощью схемы нормального потока,с левого или с правого края элемента-контейнера(также должны использовать свойство width).

Свойство clear позволяет указать, что ни один элемент, находящийся в томже элементе-контейнере, не должен касаться левой или правой стороны блока.

left -Левая сторона блока не должна касаться любого другого элемента, находящегося в том же контейнере.
right-Правая сторона блока не должна касаться любого другого элемента, находящегося в том же контейнере.
both- Ни левая, ни правая сторона блока не должны касаться любого другого элемента,находящегося в том же контейнере.
none- Элементы могут касаться любой стороны блока.

Для позиционирования колонок друг рядом с другом используются следующие три свойства CSS. width Задает ширину колонок. float Позиционирует колонки друг рядом с другом. marqin Создает интервал между колонками.. (columnlofЗ, . column2of3, . column3of3 width : ЗООрх; float : left ; margin : l Opx ; )
